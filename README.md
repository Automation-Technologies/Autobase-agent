# AutoBase Bot - Инструкция по использованию

## Структура приложения

Приложение **НЕ полностью самодостаточное**. Оно требует следующую структуру папок рядом с `.exe` файлом:

```
AutoBaseBot.exe          ← Главный файл
maFiles/                 ← Папка с данными аккаунтов (шифруется)
  ├── accounts.json      ← Данные аккаунтов
  └── *.maFile           ← Файлы Steam Guard
config/                  ← Конфигурация (создается автоматически)
  ├── config.json        ← Настройки сервера и токена
  └── proxies.json       ← Прокси для аккаунтов
logs/                    ← Логи (создается автоматически)
  └── agent.log          ← Лог работы агента
security.salt            ← Файл соли для шифрования (создается автоматически)
```

## Как использовать

### Первый запуск

1. **Скопируйте** `AutoBaseBot.exe` в нужную папку
2. **Создайте** папку `maFiles` рядом с `.exe`
3. **Поместите** ваши `.maFile` файлы в папку `maFiles`
4. **Запустите** `AutoBaseBot.exe`
5. При первом запуске появится окно: **"Придумайте новый мастер-пароль"**
   - Придумайте и запомните пароль (он нужен для всех последующих запусков)
   - После ввода пароля папка `maFiles` будет зашифрована
6. Откроется главное окно приложения

### Последующие запуски

1. **Запустите** `AutoBaseBot.exe`
2. Появится окно: **"Введите мастер-пароль"**
3. Введите ваш мастер-пароль
4. Папка `maFiles` автоматически расшифруется
5. Откроется главное окно приложения

### Завершение работы

При закрытии приложения (крестик или завершение работы):
- Папка `maFiles` **автоматически зашифруется обратно**
- Файлы получат расширение `.enc`
- Данные будут защищены

## Где хранятся данные

Все данные хранятся **рядом с `.exe` файлом** в следующих папках:

- **`maFiles/`** - Данные аккаунтов Steam (логины, пароли, maFile, cookies)
  - ⚠️ **ВАЖНО**: Эта папка шифруется мастер-паролем
  - При работе файлы расшифрованы (без `.enc`)
  - При закрытии файлы зашифрованы (с `.enc`)

- **`config/`** - Конфигурация приложения
  - `config.json` - IP сервера и токен агента
  - `proxies.json` - Прокси для аккаунтов

- **`logs/`** - Логи работы
  - `agent.log` - Основной лог

- **`security.salt`** - Файл соли для шифрования (не удаляйте!)

## Перенос на другой компьютер

1. Скопируйте **весь набор файлов**:
   - `AutoBaseBot.exe`
   - `maFiles/` (вместе с зашифрованными файлами `.enc`)
   - `config/` (если есть)
   - `security.salt` (обязательно!)
   - `logs/` (опционально)

2. На новом компьютере запустите `AutoBaseBot.exe`
3. Введите ваш **мастер-пароль**
4. Всё должно работать

## Безопасность

- ⚠️ **НЕ ТЕРЯЙТЕ мастер-пароль** - без него данные не расшифровать
- ⚠️ **НЕ УДАЛЯЙТЕ** файл `security.salt` - без него нельзя расшифровать данные
- ⚠️ **НЕ ПЕРЕМЕЩАЙТЕ** файлы из папки `maFiles` вручную во время работы приложения
- ✅ Папка `maFiles` автоматически шифруется при закрытии приложения
- ✅ Файлы остаются зашифрованными, пока приложение не запущено

## Решение проблем

### "Неверный пароль"
- Убедитесь, что используете правильный мастер-пароль
- Проверьте, что файл `security.salt` не поврежден

### "No module named ..."
- Убедитесь, что используете последнюю версию `AutoBaseBot.exe`
- Пересоберите `.exe` если проблема повторяется

### Данные не шифруются
- Проверьте права доступа к папке `maFiles`
- Убедитесь, что приложение корректно завершается (не убито через диспетчер задач)
